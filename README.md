# Sales Dashboard Analysis

This project showcases an interactive Sales Dashboard built to analyze global sales performance. The goal was to transform raw data into actionable business insights using Power BI.

## 🚀 Key Features
- **Data Modeling:** Built a clean Star Schema for efficient data processing and relationship management.
- **Interactive Visualization:** Designed a professional, clean, and cinematic dashboard covering key sales KPIs (Total Sales, Total Profit, and Total Customers).
- **Business Insights:** 
    - Analyzed sales performance across different segments, identifying the Corporate segment as the top contributor.
    - Evaluated shipping preferences and regional profitability to optimize operational decisions.
    - Conducted detailed product and customer-level analysis to highlight top performers.

## 🛠 Tools & Technologies
- **Power BI:** Data modeling, interactive dashboard creation, and advanced data visualization.

## 💡 Insights
- The **Corporate segment** is the primary driver of sales, reaching 658K.
- **Regular Air** is the most preferred shipping mode, accounting for 990K in total sales.
- Regional analysis confirms that the **West region** is a key driver of overall company profitability.
- The **Technology category** leads in sales volume (712.26K), indicating strong market demand.
